# ALU_with_random_test_generator
ALU (arithematic logic unit)
<br>
ALU performs the arithematic and logical functions based on the selction line.This random test generator 
helps in finding the errors easily.In the traditional testbench we have give inputs manually and
we have to look the waveforms to verify the output whereas in the random test generator,it generates multiple inputs cases and 
compares with the reference model to verify output
<br>
This makes our work easy and find even corner case errors.
